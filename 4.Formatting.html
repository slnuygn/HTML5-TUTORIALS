when you want to write a paragraph and want to initialize the font size, type style="font-size: 60px;" on the p tag that starts the paragraph, this paragrapgh is 60px right now and you can use whatever number you want

there are multiple formatting tags

if we write something inbetween 'strong' tags we will get a bold writing

two write in italic writing use the 'em' tag inbetween what you want to write in italic

use the 'small' tag to make a writing small inside your paragraph, as you can see it is a bit smaller

if you want to highlight a writing you use the 'mark' tag

if we want to get a writing from the web but we want the exact style and everything without html automatically reformatting it, use 'pre' tag as shown down below 

            Korkma, sönmez bu şafaklarda yüzen al sancak;
            Sönmeden yurdumun üstünde tüten en son ocak.
            O benim milletimin yıldızıdır, parlayacak;
            O benimdir, o benim milletimindir ancak.

if we want to create a subtext use 'sub' tag such as Lorem ipsum dolor sit amet consectetur adipisicing elit.
